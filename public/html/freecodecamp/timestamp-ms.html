FreeCodeCamp: Timestamp microservice

Functionality based on this site.
  • Pass a string as an URL parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties - and a message on the nature of the error.

For example, both [...]/timestamp-ms/December%2015,%202015 and [...]/timestamp-ms/1450137600 will return { "unix": 1450137600, "natural": "December 15, 2015" }
